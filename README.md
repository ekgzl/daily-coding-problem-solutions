## How to Run

Each problem is contained within its own directory. To run a specific solution:

1.  Navigate to the problem's directory:
    ```bash
    cd Problem_001
    ```
2.  Run the solution file:
    ```bash
    python solution.py
    ```
3.  To run the tests (if available):
    ```bash
    pytest
    ```
